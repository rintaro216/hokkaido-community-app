🗾 北海道旅人アプリ
===================

📱 このアプリについて
--------------------
北海道を旅行する人々のためのコミュニティアプリです。
GPS記録、投稿共有、ユーザー検索、マップ機能などを備えています。

🚀 アプリの起動方法
------------------
1. 「北海道旅人アプリを開く.bat」をダブルクリック
2. ブラウザが自動で開き、アプリが表示されます
3. もしうまく動かない場合は「アプリを再起動.bat」を実行してください

🌐 アクセスURL
--------------
http://localhost:8083

✨ 主な機能
-----------
✅ 認証システム（ゲストログイン・メールログイン）
✅ ホーム画面（投稿タイムライン・地域フィルタ）
✅ GPS記録機能（ルート記録・GPXエクスポート）
✅ 投稿作成（写真・位置情報・タグ）
✅ コミュニティ（ユーザー検索・フォロー）
✅ マイページ（プロフィール・統計・設定）
✅ 投稿詳細・コメント機能
✅ プロフィール表示
✅ ユーザー検索

⚠️ 注意事項
-----------
- Web版では一部機能（マップ、カメラ、GPS）に制限があります
- 完全な機能を利用するにはスマートフォンアプリが推奨です
- データはローカルに保存されます

🛠️ 技術情報
-----------
- React Native + Expo
- TypeScript
- ローカルストレージ（AsyncStorage）
- エラーハンドリング・オフライン対応

📞 困った時は
------------
- アプリが表示されない → 「アプリを再起動.bat」を実行
- ブラウザで http://localhost:8083 を直接開く
- ポート8083が使用中の場合は8084、8085を試す
